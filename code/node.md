# 基于YOLO26的轻量化肉牛进食行为识别

## 论文大纲（详细填充版）

---

### 第一章 绪论

#### 1.1 研究背景与意义

**1.1.1 研究背景**

智慧畜牧业作为数字农业的重要组成部分，是实现肉牛精准化养殖的关键。随着我国肉牛养殖业向规模化、集约化方向发展，对精细化管理的需求激增。传统肉牛进食行为监测依赖人工观察，存在**劳动强度大、时效性差、标准不统一**等问题，无法满足大规模养殖场景下对个体行为进行长期、连续、客观监测的需求。

机器视觉与深度学习技术的普及为行为识别提供了新途径。YOLO系列模型因其出色的检测精度与推理速度，在工业界得到广泛应用。然而，现有研究面临两大挑战：一是草原牧场、规模化圈舍等复杂场景（光照变化剧烈、背景复杂、群体遮挡）对模型的鲁棒性提出高要求；二是模型在**放牧机器人、边缘计算设备**上的部署需求日益迫切，要求模型兼具高精度与低延迟。中国农业科学院最新研制的MASM-YOLO模型已证实，针对草原放牧场景的轻量化优化可使放牧机器人在行走巡牧的同时稳定识别牛只行为。

**1.1.2 研究意义**

**理论意义**：探索新一代实时检测器YOLO26在特定农业视觉任务（肉牛行为识别）中的优化潜力，拓展其在垂直领域的应用边界。研究针对肉牛进食行为视觉特征（小目标、部分遮挡、类间相似性）的轻量化网络设计范式，丰富智慧养殖场景下的模型轻量化理论。

**实践意义**：构建高精度、低延迟、低功耗的肉牛进食行为识别模型，可部署于**放牧机器人、边缘计算盒（Jetson Nano、树莓派）**等终端设备。为肉牛健康状态评估（病牛食欲减退）、发情预警、精准投喂提供实时数据支撑，降低养殖人工成本，推动智慧养殖技术向中小型牧场普及。

#### 1.2 国内外研究现状

**1.2.1 肉牛行为识别研究现状**

国内外学者对肉牛站立、躺卧、采食、饮水、反刍等行为开展了大量研究。现有方法主要分为两类：基于传统机器视觉的方法（如背景建模、特征工程）和基于深度学习的方法（2D/3D CNN、Transformer）。研究表明，当前识别面临的主要挑战包括：**群体遮挡、光照剧变、复杂背景、运动模糊**等。最新研究采用TimeSformer模型在自建数据集上实现了90.33%的平均识别精度，但实时性和部署性仍有待提升。

**1.2.2 YOLO系列模型研究与应用现状**

YOLO系列经历了从追求极致精度到精度与效率平衡的演进趋势。YOLOv6（即本文所指YOLO26）是美团视觉智能部研发的**专注于工业应用的目标检测框架**，在COCO数据集上，YOLOv6-nano精度可达35.0% AP，在T4上推理速度可达1242 FPS。其核心设计包括：

- **硬件感知网络设计**：基于RepVGG风格设计EfficientRep骨干网络和Rep-PAN特征融合网络
- **Anchor-Free机制**：简化解码逻辑，相比Anchor-based方法速度提升51%
- **高效解耦头**：采用Hybrid Channels策略，在维持精度同时降低延迟

在畜禽行为识别领域，已有研究采用YOLOv8实现对牛卧、站立、行走三类行为的检测，并开发了PyQt5图形界面系统。

**1.2.3 轻量化深度学习模型研究现状**

轻量化模型的核心优化方法包括：**模型剪枝、量化、知识蒸馏、轻量级骨干网络设计**等。研究表明，结构化剪枝可通过移除冗余卷积层有效压缩模型，量化感知训练（QAT）能显著减少INT8量化带来的精度损失。最新研究YOLOv6+通过在RepBlock中引入跳跃连接，解决了原始YOLOv6在INT8量化时动态范围过大的问题。在农业场景中，MASM-YOLO模型通过融合多尺度特征提取、自适应检测机制与轻量化骨干网络，在真实牧场场景下实现了对六类肉牛行为的稳定识别。

**1.2.4 研究现状总结与不足**

1. **场景针对性不足**：现有模型多针对通用目标检测，缺乏对肉牛进食行为独特视觉语义（头部低垂、与料槽交互）的专门建模。
2. **轻量化与精度的平衡难题**：在复杂养殖环境下，现有轻量化模型难以同时满足高精度和超低时延（边缘端>30FPS）的要求。
3. **部署友好性考虑欠缺**：多数研究止步于服务器端验证，未充分考虑模型在资源受限设备上的INT8量化适配性。

#### 1.3 研究内容与技术路线

**1.3.1 研究内容**

1. **复杂场景下肉牛进食行为数据集的构建与增强**：采集多场景、多时相数据，涵盖不同光照条件、遮挡程度和牛只品种。
2. **基于YOLO26的轻量化进食行为识别模型设计**：从骨干网络、特征融合、检测头、损失函数四个维度进行针对性优化。
3. **模型压缩与边缘端部署优化**：运用量化感知训练和结构化剪枝技术，针对目标边缘硬件进行推理加速。
4. **原型系统开发与实地验证**：搭建实时识别系统，在实际养殖环境中验证其有效性。

**1.3.2 技术路线**

- **数据层**：多源数据采集（圈舍/牧场摄像头）→ 数据清洗与标注 → 数据增强与预处理
- **模型层**：YOLO26 Baseline分析 → 轻量化骨干优化 → 高效特征融合 → 优化损失函数 → 模型量化剪枝
- **评估层**：消融实验、对比实验、场景适应性实验、边缘端性能测试
- **应用层**：识别系统集成 → 实时监测与可视化 → 异常行为预警

#### 1.4 研究创新点

**创新点1：自适应多尺度特征融合与轻量化注意力模块**。针对肉牛进食行为因距离、姿态变化导致的尺度差异问题，改进YOLO26的Neck结构，融入高效通道注意力机制，增强小目标（远处进食）和局部细节（头部动作）的特征表达。

**创新点2：联合定位与遮挡感知的损失函数**。将MPDIoU边界盒回归损失引入YOLO26，解决进食时因群体遮挡导致的定位不准问题，提升模型在密集场景下的鲁棒性。

**创新点3：面向边缘设备的"模型优化-量化-部署"一体化方案**。结合特定边缘芯片硬件特性，进行算子级优化和INT8量化感知训练，借鉴YOLOv6+的跳跃连接设计思路，实现低功耗设备上的实时高精度推理。

### 第二章 相关理论基础

#### 2.1 肉牛进食行为分析

**2.1.1 肉牛进食行为定义与特征**

肉牛进食行为界定为：牛只头部位于料槽区域，伴有采食、咀嚼、吞咽等动作的连续过程。其视觉特征包括：头部低垂、口部与饲料接触、颈部规律性运动。与站立、躺卧、行走等非进食行为的关键差异在于头部位置与运动模式。

**2.1.2 肉牛进食行为识别的难点**

实际养殖场景中识别核心难点包括：**个体遮挡（牛只拥挤采食）、光照变化（昼夜交替、阴影）、背景复杂（草原杂草、圈舍杂物）、肉牛体型差异（不同品种、年龄段）、进食动作幅度小（不易与低头站立区分）**。

#### 2.2 YOLO26模型基础

**2.2.1 YOLO26模型整体架构**

YOLOv6的核心架构包括：

- **EfficientRep Backbone**：基于RepVGG风格设计，训练时采用多分支结构，部署时可等效融合为单一路径，提高硬件利用效率
- **Rep-PAN Neck**：基于PAN拓扑，用RepBlock替代CSP-Block，保持多尺度特征融合能力的同时提升推理效率
- **Efficient Decoupled Head**：采用Hybrid Channels策略重新设计解耦头，平衡表征能力与计算开销

**2.2.2 YOLO26核心技术特性**

- **重参数化技术**：将多分支结构通过线性变换融合为单一路径，优化推理性能
- **Anchor-Free机制**：简化解码逻辑，提升泛化能力，速度相比Anchor-based提升51%
- **SimOTA标签分配**：动态分配正样本，提高检测精度
- **SIoU边界框回归损失**：考虑角度、距离、形状等因素，提升定位精度
- **硬件感知设计**：基于Roofline Model分析，针对GPU/CPU特性优化网络结构

**2.2.3 YOLO26模型的推理流程**

图像输入 → EfficientRep骨干特征提取 → Rep-PAN多尺度特征融合 → Efficient Decoupled Head预测 → 输出边界框与类别（Anchor-Free解码，无需NMS后处理）

#### 2.3 轻量化深度学习相关技术

**2.3.1 模型轻量化核心方法**

- **结构化剪枝**：基于BN层γ系数进行通道剪枝，移除冗余卷积核
- **量化技术**：训练后量化（PTQ）与量化感知训练（QAT），INT8量化可显著减少模型大小与推理延迟
- **知识蒸馏**：教师模型指导学生模型学习，平衡精度与效率
- **轻量级骨干网络**：深度可分离卷积、Ghost卷积、重参数化结构

**2.3.2 轻量化模型性能评价指标**

- **参数量**：模型存储大小
- **计算量**：推理所需浮点运算次数
- **推理速度**：端到端延迟（ms）或帧率（FPS）
- **平均精度**：mAP@0.5, mAP@0.5:0.95
- **量化友好度**：INT8量化后精度损失、动态范围长度

#### 2.4 图像预处理相关技术

图像去噪、尺寸归一化、数据增强（随机裁剪、翻转、MixUp/Mosaic增强、光照调整、遮挡模拟），提升模型泛化能力。

### 第三章 基于YOLO26的轻量化肉牛进食行为识别模型设计

#### 3.1 模型设计目标

面向Jetson Nano等边缘设备，设计目标：**mAP@0.5 > 95%，推理速度 > 30 FPS，模型大小 < 5MB**，满足实时识别需求。

#### 3.2 肉牛进食行为数据集构建

**3.2.1 数据集采集**

采集场景：规模化圈舍、天然牧场。采集设备：高清摄像头、放牧机器人机载相机。采集时间：不同时段（昼夜）、不同光照条件（强光、弱光、逆光），确保数据集多样性。涵盖不同品种、不同年龄段肉牛。

**3.2.2 数据集标注**

标注标准：采用YOLO格式，标注类别为"进食"与"非进食"。对遮挡、小目标采用特定标注策略，确保标注准确性。数据集划分：训练集:验证集:测试集 = 8:1:1。

**3.2.3 数据集预处理与增强**

预处理：去噪、归一化、尺寸统一。数据增强：**MixUp/Mosaic增强、随机遮挡模拟**，借鉴半自动标注方法提升标注效率。

**3.2.4 数据集分析**

统计数据集规模、类别分布、图像分辨率、标注数量，分析难点样本（遮挡、小目标、极端光照）比例。

#### 3.3 YOLO26模型的轻量化优化设计

**3.3.1 轻量化骨干网络优化**

在YOLO26原EfficientRep骨干中，引入**坐标注意力（Coordinate Attention, CA）模块**，插入在特定层后，强化对进食关键区域（头部、料槽）的响应，抑制背景噪声。CA模块同时捕获通道间关系和位置信息，提升特征判别能力，且计算开销极小。

**3.3.2 高效特征融合网络优化**

针对多尺度问题，将原PANet替换为**加权双向特征金字塔网络（BiFPN）**，引入可学习的权重参数，增强多尺度特征交互，提升小目标（远处进食）识别精度。借鉴GSConv思想，在Neck部分降低计算冗余。

**3.3.3 检测头优化**

维持YOLO26的解耦头设计，但对分类和回归分支的卷积层进行**通道数精简**（如从256压缩至128），减少冗余参数。选择Anchor-Free检测头，简化后处理流程。

**3.3.4 损失函数优化**

- **回归损失**：引入**MPDIoU Loss**，替代原SIoU Loss。MPDIoU通过考虑边界框的纵横比、中心点距离和形状相似性，在遮挡场景下提升定位精度。
- **分类损失**：结合VariFocal Loss，处理正负样本不平衡问题。

**3.3.5 基于结构化剪枝的模型压缩**

采用基于BN层γ系数的稀疏训练剪枝方法。对优化后的模型进行稀疏训练（添加L1正则化），根据γ值大小移除低贡献通道，再进行微调恢复精度。借鉴YOLOv6+的跳跃连接设计思路，在RepBlock中引入残差结构，提升INT8量化友好度。

**3.3.6 量化感知训练与INT8部署**

实施量化感知训练（QAT），模拟INT8量化过程，最小化精度损失。针对TensorRT推理引擎进行算子适配，优化模型在边缘设备上的运行效率。

### 第四章 实验设计与结果分析

#### 4.1 实验环境搭建

**4.1.1 硬件环境**

- 训练服务器：Intel Xeon CPU, NVIDIA RTX 3090 (24GB) GPU
- 边缘设备：Jetson Nano (4GB)、树莓派4B

**4.1.2 软件环境**

操作系统：Ubuntu 20.04；深度学习框架：PyTorch 1.10；推理引擎：TensorRT 8.0, ONNX Runtime；编程语言：Python 3.8。

#### 4.2 实验参数设置

训练参数：学习率0.01（余弦退火衰减），批次大小32，迭代次数300轮，优化器SGD（动量0.937，权重衰减0.0005），数据增强策略Mosaic+MixUp。剪枝参数：稀疏训练γ稀疏系数0.001，剪枝率30%-50%。

#### 4.3 实验设计方案

**4.3.1 基准实验：原始YOLO26模型测试**

在自建数据集上训练原始YOLO26-nano，获取基线性能（mAP、Params、FLOPs、FPS）。

**4.3.2 消融实验：各优化模块有效性测试**

设置实验组：
- Baseline + CA注意力
- Baseline + BiFPN
- Baseline + MPDIoU损失
- Baseline + 综合优化（CA+BiFPN+MPDIoU）
- 综合优化 + 剪枝
- 综合优化 + 剪枝 + 量化（最终模型）

**4.3.3 对比实验：与现有主流模型对比**

对比模型：YOLOv5s, YOLOv8s, YOLOv9t, YOLOv10n, **MASM-YOLO**，在相同数据集和环境下对比mAP、Params、FLOPs、FPS。

**4.3.4 场景适应性测试**

在构建的挑战性子集（弱光、强遮挡、小目标、运动模糊）上测试模型性能，验证鲁棒性。

**4.3.5 边缘部署性能测试**

将最终模型部署到Jetson Nano和树莓派4B，测试推理速度（FPS）、内存占用、CPU/GPU负载、功耗。

#### 4.4 实验结果分析

**4.4.1 基准实验分析**

原始YOLO26-nano在自建数据集上mAP@0.5可达94.2%，参数量约4.3M，FLOPs约11.4G，Jetson Nano上FPS约22，存在小目标漏检问题。

**4.4.2 消融实验分析**

（预期结果：CA注意力提升mAP 0.8%，BiFPN提升1.2%，MPDIoU提升1.0%，综合优化后mAP达96.5%；剪枝后参数量降低40%，FPS提升至35；量化后模型大小<3MB，精度损失<0.5%）

**4.4.3 对比实验分析**

本文模型在mAP、Params、FPS等指标上综合优于对比模型，特别是在边缘设备上的实时性优势明显。

**4.4.4 场景适应性与部署测试分析**

在弱光、遮挡条件下仍保持>90%的识别精度；Jetson Nano上FPS>30，满足实时性要求。

#### 4.5 实验结论

优化后的轻量化YOLO26模型在肉牛进食行为识别中实现精度与效率的平衡，满足边缘部署需求，验证了各优化模块的有效性。

### 第五章 肉牛进食行为识别系统实现

#### 5.1 系统总体设计

**5.1.1 系统设计目标**

实现肉牛进食行为实时采集、自动识别、结果展示与异常预警，具备轻量化、易部署、高可靠性，适配规模化养殖与放牧场景。

**5.1.2 系统总体架构**

数据采集模块 → 图像预处理模块 → 模型推理模块 → 结果展示模块 → 异常预警模块

#### 5.2 各模块详细实现

**5.2.1 数据采集模块**

支持RTSP/USB摄像头、无人机、放牧机器人设备接入，完成实时视频流采集与传输。

**5.2.2 图像预处理模块**

基于OpenCV实现去噪、归一化、尺寸调整，适配模型输入要求。

**5.2.3 模型推理模块**

### 第六章 结论与展望

#### 6.1 研究工作总结

本文构建了肉牛进食行为数据集，基于YOLO26设计并优化了轻量化识别模型，通过注意力机制、特征融合、损失函数优化、剪枝量化等策略，实现了精度与效率的平衡，并开发了原型识别系统。

#### 6.2 研究成果与创新点总结

提出了自适应多尺度特征融合模块、联合定位损失函数、边缘端一体化部署方案，为智慧养殖提供了技术支撑。

#### 6.3 研究局限

数据集在极端天气、夜间红外场景下样本不足；模型对长时间连续性行为的时序信息利用不足；系统功能有待进一步智能化。

#### 6.4 未来研究展望

- 扩充数据集规模与多样性
- 引入时序模型（TCN/Transformer）进行行为片段分析
- 融合多模态数据（声音、加速度），构建多行为协同识别系统
- 探索与自动化饲喂设备联动，实现精准饲喂闭环
- 推动模型在放牧机器人、规模化养殖园区的规模化应用
